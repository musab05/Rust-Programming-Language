### A2
Issues:
- `name = "my app"` — crate names can't have spaces. Use `"my-app"` or `"my_app"`.
- `version = "1.0"` — should be `"1.0.0"` (full semver).
- `serde = 1.0` — version must be a string: `serde = "1.0"`.

```toml
[package]
name = "my-app"
version = "1.0.0"

[dependencies]
serde = "1.0"
```

## Section C

### A7 — Serialize a struct
```rust
use serde::{Serialize, Deserialize};

#[derive(Serialize, Deserialize, Debug)]
struct Book {
    title: String,
    author: String,
    pages: u32,
    rating: f64,
}

fn main() {
    let book1 = Book {
        title: String::from("The Rust Programming Language"),
        author: String::from("Steve Klabnik"),
        pages: 560,
        rating: 4.8,
    };

    let book2 = Book {
        title: String::from("Programming Rust"),
        author: String::from("Jim Blandy"),
        pages: 738,
        rating: 4.7,
    };

    // Serialize to JSON
    let json1 = serde_json::to_string_pretty(&book1).unwrap();
    let json2 = serde_json::to_string_pretty(&book2).unwrap();
    println!("Book 1:\n{json1}\n");
    println!("Book 2:\n{json2}\n");

    // Deserialize from JSON
    let json_str = r#"{
        "title": "Rust in Action",
        "author": "Tim McNamara",
        "pages": 456,
        "rating": 4.6
    }"#;
    let book3: Book = serde_json::from_str(json_str).unwrap();
    println!("Deserialized: {book3:?}");
}
```

### A8 — Library design
```rust
// src/lib.rs

/// Adds two numbers.
pub fn add(a: f64, b: f64) -> f64 {
    a + b
}

/// Subtracts b from a.
pub fn subtract(a: f64, b: f64) -> f64 {
    a - b
}

/// Multiplies two numbers.
pub fn multiply(a: f64, b: f64) -> f64 {
    a * b
}

/// Divides a by b. Returns Err if b is zero.
pub fn divide(a: f64, b: f64) -> Result<f64, String> {
    if b == 0.0 {
        Err(String::from("Division by zero"))
    } else {
        Ok(a / b)
    }
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn test_add() {
        assert_eq!(add(2.0, 3.0), 5.0);
        assert_eq!(add(-1.0, 1.0), 0.0);
    }

    #[test]
    fn test_subtract() {
        assert_eq!(subtract(10.0, 3.0), 7.0);
    }

    #[test]
    fn test_multiply() {
        assert_eq!(multiply(4.0, 5.0), 20.0);
    }

    #[test]
    fn test_divide() {
        assert_eq!(divide(10.0, 2.0), Ok(5.0));
        assert!(divide(10.0, 0.0).is_err());
    }
}
```

### A9 — Cargo commands
```bash
# 1. Create a new binary project
cargo new my_app

# 2. Add rand as a dependency
cargo add rand

# 3. Build in release mode
cargo build --release

# 4. Run tests with "math" in their name
cargo test math

# 5. View dependency tree
cargo tree

# 6. Format all source code
cargo fmt

# 7. Run the linter
cargo clippy

# 8. Generate and open documentation
cargo doc --open
```
